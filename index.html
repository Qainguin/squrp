Browser-Native Compression

Compress files in the
browser, blazing fast

A lightweight library for zipping and unzipping file maps using the native CompressionStream API. Zero dependencies, full TypeScript support.

Get Started View Benchmarks
example.ts 
import { zip, unzip } from 'squrp';

// Create a file map with text or binary content
const files = new Map([
  ['hello.txt', 'Hello, World!'],
  ['data.bin', new Uint8Array([1, 2, 3, 4, 5])],
]);

// Compress with a single line
const compressed = await zip(files);

// Decompress just as easily
const unzipped = await unzip(compressed);

Everything you need for browser compression

Simple, powerful, and designed for modern web applications.

Blazing Fast

Uses the native CompressionStream API for maximum performance. No WASM, no native modules.

Multiple Formats

Support for gzip, brotli, and deflate compression formats. Choose the right balance of speed and size.

Full TypeScript

Complete type definitions with JSDoc documentation. Works great with IDE autocomplete.

Flexible I/O

Works with Uint8Array, Blob, or Base64. Perfect for any use case from downloads to API calls.

Error Handling

Detailed error codes and messages. Handle corruption, truncation, and validation errors gracefully.

Mixed Content

Store both text and binary files in the same archive. Text decoded as strings, binary as Uint8Array.

Basic Usage

import { zip, unzip } from 'squrp';

// Objects work too!
const files = {
  'config.json': '{"version": 1}',
  'data.bin': new Uint8Array([1, 2, 3]),
};

const compressed = await zip(files);
const unzipped = await unzip(compressed);

Choose Your Format

import { zip } from 'squrp';

// Best compatibility (default)
await zip(files);
await zip(files, { compression: 'gzip' });

// Best compression ratio
await zip(files, { compression: 'brotli' });

// Fastest compression
await zip(files, { compression: 'deflate' });

Blob Support

import { zipToBlob, unzipFromBlob } from 'squrp';

// Create a Blob for download/upload
const blob = await zipToBlob(files);
await fetch('/api/upload', {
  method: 'POST',
  body: blob,
});

// Unzip from a Blob
const download = await fetch('/download');
const blob = await download.blob();
const unzipped = await unzipFromBlob(blob);

Base64 for JSON APIs

import { zipToBase64, unzipFromBase64 } from 'squrp';

// Send compressed data in JSON
const base64 = await zipToBase64(files);
await fetch('/api/save', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ data: base64 }),
});

// Load and unzip from JSON
const loadResponse = await fetch('/api/load').then(r => r.json());
const unzipped = await unzipFromBase64(loadResponse.data);
